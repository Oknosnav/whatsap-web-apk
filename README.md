# WhatsApp Web APK

Optimized Android app for WhatsApp Web with clean UI and fast performance.

## Features
- ⚡ Hardware accelerated WebView
- 🚫 Ad and tracking blocker
- 💾 Smart caching
- 📲 Progress indicator
- 🔐 Secure cookie management

## Build Instructions
1. Clone: `git clone https://github.com/Oknosnav/whatsap-web-apk.git`
2. Open in Android Studio
3. Build → Build APK(s)
